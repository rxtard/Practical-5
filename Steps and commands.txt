PRACTICAL NO: 05 
Aim: Raspberry Pi GPS Module interfacing
Wire Connection:
GPS Module		GPIO pin
VCC				Pin 2
GND				Pin 6
Tx				Pin 10

Jumper cable type: Female to Female

Step1 Create Venv
sudo apt update
sudo apt install python3-venv -y
python3 -m venv myenv
source myenv/bin/activate

---

Step2 Enable UART
sudo nano /boot/config.txt

Add lines:
dtparam=spi=on
dtoverlay=pi3-disable-bt
core_freq=250
enable_uart=1
force_turbo=1

Save:
CTRL+X
Y
ENTER

---

Step3 Disable Serial Console
sudo systemctl stop serial-getty@ttyS0.service
sudo systemctl disable serial-getty@ttyS0.service
sudo systemctl enable serial-getty@ttyAMA0.service

sudo reboot

---

Step4 Install packages
sudo apt install minicom -y
pip install pynmea2 --break-system-packages
pip install pyserial --break-system-packages
---

Step5 Test GPS
sudo cat /dev/ttyS0
or
sudo cat /dev/serial0
---
Step6 Python Code (gps.py)
import serial
import time
import pynmea2
port = "/dev/serial0"
ser = serial.Serial(port, baudrate=9600, timeout=0.5)
while True:
    newdata = ser.readline().decode("utf-8", errors="ignore")
    if newdata.startswith("$GPRMC"):
        try:
            newmsg = pynmea2.parse(newdata)
            lat = newmsg.latitude
            lng = newmsg.longitude
            print("Latitude =", lat, "Longitude =", lng)
        except pynmea2.ParseError:
            continue
